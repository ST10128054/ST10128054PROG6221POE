This ReadMe file will explain how to use this application.

After launching the app you will be greeted with a menu where you can choose from the following options.
-Enter ingredients and steps
-Scale measurements
-Reset measurements
-Display recipe
-Clear recipe
-Exit

Entering the number 1 will allow you to enter details of the recipe.
Entering number 2 will let you pick what factor to scale the quantities by.
Entering number 3 will reset those scaled quantities back to the original numbers.
Entering number 4 will display the details of the recipe.
Entering 5 will clear the details of the recipe.
Entering 6 twice will exit the app.

For each function after completion you will be taken back to the main menu to choose another function.
